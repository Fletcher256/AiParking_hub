# HiEuler Official Linux Firmware

Generated from the official `HiEuler_PI_Firmware_Building` package with:

```bash
./Generate_Image.sh -g -os Linux
```

Use `burn_table_4G.xml` in ToolPlatform for the current 4 GB board. The `burn_table_4G_1rank.xml` alternate is included only in case the board is confirmed to be a 1-rank DDR variant.
